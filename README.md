# 💬 Real-Time Chat Application

## 👩‍💻 About the Project
This is a Real-Time Chat Application that allows users to communicate instantly using a client-server architecture. It demonstrates real-time communication using WebSockets/Socket programming.

---

## 🚀 Features
- ⚡ Instant real-time messaging between users  
- 👥 Multiple user support  
- 🔔 Live message updates without page refresh  
- 🟢 User connection status (online/offline) *(if implemented)*  
- 💬 Simple and responsive chat interface  

---

## 🏗️ Project Structure
      ---

## 🛠️ Technologies Used
- HTML, CSS, JavaScript  
- WebSockets / Socket.io  
- Python (Flask / Socket Server) or Node.js (Express)  
- Client-Server Architecture  

---

## ⚙️ How It Works
1. Server is started on a specific port  
2. Client connects to the server using WebSockets  
3. When a user sends a message, it is sent to the server  
4. Server broadcasts the message to all connected clients instantly
